# EDSDA import structure folder #

```
/edsda-import:
  /client:
  /server:

```
- `client`: User interface for importing data,...
- `server`: Server handling import file from user. 

### For running this file ###

* Quick summary
* Version

### How do I get set up? ###
- Access to client folder
- `cd client`
- `npm install`
- `npm start`

- Access to server folder
- `cd client`
- `npm install`
- `npm start`
